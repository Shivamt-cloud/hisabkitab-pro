## ğŸ¯ Key Points

### Device Limits
1. **Company-Based:** Device limits apply to the ENTIRE company, not per user
2. **Shared Pool:** All users in a company share the device limit
3. **Device Types:** Mobile, Laptop, Desktop, Tablet are all counted equally
4. **Registration:** Devices are automatically registered on login
5. **Management:** Admin can view and remove devices in System Settings

### User Limits
1. **Company-Based:** User limits apply to the ENTIRE company
2. **Enforcement:** System blocks user creation when limit is reached
3. **Upgrade Required:** To add more users, company must upgrade plan
4. **Admin Override:** Admin can manage users but still subject to limits

### Plan Selection
1. **During Registration:** User selects plan when registering
2. **Stored with Company:** Plan is stored with company record
3. **Applied to Company:** All users in company are subject to company's plan limits
4. **Upgrade Path:** Basic â†’ Standard â†’ Premium
