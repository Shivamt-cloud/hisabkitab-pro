## Detailed Description (for README/GitHub About section)

```
HisabKitab-Pro is a modern, offline-first inventory management system designed for small to medium businesses. The application provides comprehensive features for managing products, sales, purchases, customers, suppliers, and inventory with real-time stock tracking, detailed analytics, and robust reporting capabilities.

Key Features:
• Complete inventory management with real-time stock tracking
• Sales and purchase management with GST support
• Customer and supplier management
• Stock adjustments and low stock alerts
• Comprehensive analytics and reporting dashboard
• User management with role-based permissions
• Offline-first architecture (works without internet)
• Barcode generation and management
• Audit logging for all operations
• Automatic backup system
• Multi-device ready (with cloud sync support)

Tech Stack: React, TypeScript, Vite, Tailwind CSS, IndexedDB, React Router

Deployed at: hisabkitabpro.com
```
