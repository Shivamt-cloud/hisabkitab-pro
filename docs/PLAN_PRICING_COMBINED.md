# HisabKitab Pro – Complete Plan Pricing (Combined)

Single reference for all plans: **Mobile**, **Desktop**, and **Combo (Mobile + Desktop)**.  
**Starter** includes Purchase Report. **3-year** pricing has ~25% discount.

---

## Access types

| Type | Description |
|------|-------------|
| **Mobile** | Phone/tablet only (mobile browser or app). Entry pricing from ₹55/mo. |
| **Desktop** | PC/laptop only (desktop browser). |
| **Combo** | Both; same login, all devices. Full plan price. |

**3-year discount:** Pay ~2.25× one-year price for 3 years (~25% off vs 1-year × 3).
