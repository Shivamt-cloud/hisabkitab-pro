# HisabKitab Pro – Plan Pricing by Platform (Mobile / Desktop / Combo)

## Logic used

- **Starter Mobile:** ₹55/mo, ₹649/yr (as requested).
- **Desktop** (same plan): ~1.54× Mobile yearly.
- **Combo** (same plan): ~2× Mobile yearly.
- **Higher plans:** Mobile steps ~₹44–50/mo per tier; Desktop & Combo scale similarly.
- **3-year:** 1-year × 2.25 (about 25% off vs paying 1-year × 3).
