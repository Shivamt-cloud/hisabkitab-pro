# HisabKitab Pro – Plan Pricing by Platform (Mobile / Desktop / Combo)

**Bifurcation:** Every plan is available in three access types:
- **Mobile** – Use on phone/tablet only (mobile browser or app).
- **Desktop** – Use on PC/laptop only (desktop browser).
- **Combo (Mobile + Desktop)** – Use on both; same login, all devices.

**3-year discount:** ~25% off (pay equivalent of ~2.25 years for 3 years).

## Pricing logic (for reference)

- **Combo** = full plan price (all devices); same as your current “single price” per plan.
- **Desktop** ≈ 75–85% of Combo (desktop-only access).
- **Mobile** = entry pricing; starts at ₹55/mo (₹649/yr) for Starter. Desktop & Combo unchanged.
- **3-year** ≈ 2.25× the 1-year price (~25% off vs pay 1-year × 3).

Combo is always less than Mobile + Desktop bought separately, so there is a clear benefit to the bundle.
